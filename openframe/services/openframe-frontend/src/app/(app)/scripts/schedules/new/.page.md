A Next.js page component at `/scripts/schedules/new` that renders the schedule creation interface.

## Key Components

- **NewSchedulePage**: Default exported React component that serves as the page entry point
- **ScheduleCreateView**: Core component handling the schedule creation functionality and form interface (renders in create mode when no `scheduleId` is passed)

## Usage Example

This page is automatically routed by Next.js based on its file location (renamed from `/scripts/schedules/create` during the create-route alignment; the old path is remapped in `not-found.tsx`).

```typescript
// Rendered page structure
<ScheduleCreateView />
// Schedule creation form and controls
```

The `'use client'` directive indicates this is a client-side component that can use React hooks and browser APIs. The page is statically exportable — it carries no route segment config.
