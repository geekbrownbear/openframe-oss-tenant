Top-level module declaration for the `utils` namespace, exposing small helper modules used across the client.

## Key Components

- **`windows_helpers`** *(Windows-only)* - Wide-string conversion (`to_wide`, `wcslen`), command-line construction (`build_command_line`), and HKLM autorun registry helpers (`register_autorun`, `unregister_autorun`).
- **`version_comparator`** - Semantic-version comparison utility that normalizes incomplete versions before comparing.

## Usage Example

```rust
use crate::utils::version_comparator::VersionComparator;

#[cfg(target_os = "windows")]
use crate::utils::windows_helpers::register_autorun;

let comparator = VersionComparator;
assert!(comparator.compare("1.2", "1.2.0").is_eq());

#[cfg(target_os = "windows")]
register_autorun("my-app", r"C:\path\to\app.exe", &[])?;
```
